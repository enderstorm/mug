= Install

```
curl https://github.com/seges/mug/raw/master/setup.sh | sudo sh
```

= Run

== Existing project

```
cd <project_dir>
mug
```

== New project

When you want to create a new project and you want to utilize the environment:

```
cd <workspace_dir>
mug <type>
```

Where *type* is one of:

* backend-scala
* frontend-javascript
